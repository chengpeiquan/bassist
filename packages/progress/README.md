Simple slim progress bars base on NProgress.

## Usage

With npm(or yarn, or pnpm):

```bash
npm install @bassist/progress
```

In `.js` / `.ts` or other files:

```ts
import progress from '@bassist/progress'

progress.start();
progress.done();
```
